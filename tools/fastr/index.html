FASTR: Reimagining FASTQ via Compact Image-Inspired Representation

International Symposium on Molecular Biology (ISMB), 2026

Authors

Adrian Tkachenko, Sepehr Salem, Ayotomiwa Ezekiel Adeniyi, Zülal Bingöl, Mohammed Nayeem Uddin, Akshat Prasanna, Alexander Zelikovsky, Serghei Mangul, Can Alkan, Mohammed Alser

ALSER Lab, Georgia State University · Bilkent University · Sage Bionetworks

Abstract

High-throughput sequencing (HTS) enables population-scale genomics but generates massive datasets that strain storage, transfer, and analysis pipelines. FASTQ, the dominant sequencing format for over two decades, stores one byte per nucleotide and one byte per quality score, resulting in inefficient I/O and significant redundancy.

We introduce FASTR, a lossless, computation-native successor to FASTQ that encodes each nucleotide together with its quality score into a single byte using implicit partitioned range encoding. FASTR reduces file size by at least 2× while remaining fully reversible and directly usable for downstream analysis without decompression.

FASTR enables machine-learning-ready representations and integrates seamlessly with existing tools such as minimap2. We provide a massively parallel FASTQ–FASTR conversion ecosystem and demonstrate improved compression ratios, faster runtimes, and reduced data movement across Illumina, HiFi, and Oxford Nanopore datasets.

Key Contributions

  • Lossless successor to FASTQ with implicit base–quality encoding
  • ≥2× reduction in storage footprint
  • Massively parallel FASTQ–FASTR conversion
  • Drop-in compatibility with existing genomics tools
  • Machine-learning-ready numeric and image-inspired format
